# Background #
##############

-Info is stored in the following ways:
	-Objects : Specific things / proper nouns ( ex: Clayton, Computer, That slice of pizza, etc... )
	-Words   : General words / global definitions ( ex: hot, cube, apple (fruit), etc... )
	-Events  : Sentences that describe interactions between different objects / words ( ex : Clayton [obj] - ate [word] - pie [word]  )
